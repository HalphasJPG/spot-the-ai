# Spot the AI

Welcome to the **Spot the AI Project**! This is a browser-based, AI-versus-Human guessing game developed as a group project. The game challenges players to look at an image and correctly determine whether it was generated by Artificial Intelligence or captured/created by a human.

🚀 **Zero Installation Required:** The application runs entirely in any modern web browser and is fully hosted and deployed on Amazon Web Services (AWS).

## 🎮 Game Modes & Features

The Antigravity Project offers distinct ways to play depending on whether you want to test your own skills or challenge your friends:

* **Single-Player Mode (Core Experience):** Race against the clock or play at your own pace to see how accurately you can distinguish real human creativity from AI-generated generations across randomized image sets.
* **Multiplayer Mode (Feature):** Host or join a lobby to compete in real-time with your friends! Go head-to-head to see who can spot the AI fakes fastest and climb the match leaderboard.
* **Responsive Web Design:** Fully optimized for smooth gameplay across desktop monitors, tablets, and mobile devices.
* **Instant Cloud Delivery:** Powered by a lightweight frontend and AWS for fast image loading and seamless lobby synchronization.

## 🛠️ Local Development (For Contributors)

If you are a member of the group or a contributor looking to run and test the game locally, follow these steps:

### Prerequisites
* Node.js installed locally.

### Installation & Setup

1. **Clone the repository:**
   git clone https://github.com/HalphasJPG/Antigravity-Project.git
   cd Antigravity-Project

2. **Install dependencies:**
   npm install

3. **Start the local development server:**
   npm run dev
   # or
   npm start

4. Open http://localhost:3000 in your browser to test your local instance of the game.

## ☁️ Deployment Pipeline

Our automated deployment pipeline handles the heavy lifting when updating the live game:
1. **Push:** Code changes are pushed or merged into the `main` branch.
2. **Build:** GitHub Actions runs build optimizations to pack the frontend assets cleanly.
3. **Deploy:** The assets are automatically synced to our AWS environment, making updates live immediately without disconnecting players.
