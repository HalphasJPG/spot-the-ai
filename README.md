# 🌌 Project Antigravity Web App

Welcome to our group project repository! This web-based application was built using **Google Antigravity**, leveraging autonomous multi-agent AI workflows to design, code, and test the platform. 

🚀 **Zero Installation Required:** The final application is built for the web and runs entirely in the browser, fully hosted and deployed on Amazon Web Services (AWS).

## 🚀 Project Overview

The objective of this project was to collaborate as a team alongside **Google Antigravity's agent-first ecosystem** to rapidly scaffold, develop, and deploy a fully functional web application. By utilizing Antigravity's asynchronous multi-agent orchestration, we were able to run parallel development tracks (frontend, backend API, and testing) smoothly.

### Key Features
* **Built with Agentic AI:** Developed using Google Antigravity and Gemini 3.5 Flash multi-agent teams.
* **Instant Browser Access:** No user downloads, local setup, or configurations required to run the app.
* **Automated Verification:** End-to-end functionality was verified autonomously using Antigravity browser testing artifacts before deployment.
* **Cloud-Native Hosting:** Fully optimized for high availability on AWS.

## ⚙️ Development Workflow

Because this project was developed using Google Antigravity, our workflow shifted from manual coding to goal-oriented orchestration:

1. **Prompting & Planning:** We utilized Antigravity's `/grill-me` command to let the agent clarify requirements before building a concrete Task Plan.
2. **Parallel Agent Execution:** Antigravity split tasks among subagents to handle asset generation, frontend layout, and code verification simultaneously.
3. **Local Validation:** The AI automatically launched local servers and utilized built-in browser testing tools to verify functionality, producing media artifacts of successful runs.

## 🛠️ Local Contributor Setup

If you want to pull down this repository and work on it using Google Antigravity locally:

### Prerequisites
* Google Antigravity Desktop App or Antigravity CLI installed.
* Node.js installed locally.

### Steps
1. **Clone the repository:**
   git clone https://github.com/YOUR-USERNAME/YOUR-REPO-NAME.git
   cd YOUR-REPO-NAME

2. **Initialize the Antigravity Project:**
   Open the folder inside Google Antigravity to let the workspace sync permissions and environment variables.

3. **Install and Run:**
   npm install
   npm run dev
